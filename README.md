# JOBTRIX: A Job Portal Application

**JOBTRIX** is an intelligent job search and recruitment platform designed to bridge the gap between job seekers and employers using machine learning. Unlike traditional portals, JOBTRIX analyzes a user's unique profile and skillset to automatically recommend jobs where they are most likely to excel.

## 🚀 Key Features
* **Automated Recommendations:** Implements the **XGBoost algorithm** to match individuals with suitable job openings based on their skills.
* **Live Data Scraping:** Uses **Beautiful Soup** to scrape real-time job listings from portals like TimesJobs, ensuring the database is always current.
* **RESTful Architecture:** Built using **Flask REST API** for a scalable, stateless, and efficient backend.
* **Resume Builder:** An integrated feature that allows users to generate and download professional resumes in PDF format.
* **Skill Assessment:** Includes a quiz module (Python, Java, HTML, etc.) to help users test their proficiency and prepare for interviews.
* **Admin Dashboard:** A centralized interface for administrators to manage users, verify registrations, and view applied jobs.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python, Flask REST API
* **Database:** MySQL (Storage) & SQLite3 (Local)
* **Machine Learning:** XGBoost Algorithm
* **Web Scraping:** Beautiful Soup, Requests

## 📊 System Architecture
